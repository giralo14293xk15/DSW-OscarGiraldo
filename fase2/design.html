Propósito del sitio

El propósito de este sitio es fomentar el aprendizaje de las Etiquetas semánticas y CSS Grid en HTML correspondiente a la fase 2 del curso de Diseño de Sitios Web

Tema 1 - Etiquetas semánticas

Las etiquetas semánticas son unidades léxicas las cuales ayudan a definicr básicamente la estructura de una página web, permitiendo que estas sean mejor indexadas por los diferentes buscadores. Podemos nombrar las principales (header, article, section, aside, footer, nav, figure, time entre otras).

Blanco Escoda, X. (2010). Etiquetas semánticas de hecho como género próximo en la definición lexicográfica. Recuperado de: https://roderic.uv.es/bitstream/handle/10550/30287/159.pdf?sequence=1

Tema 2 - CSS Grid

CSS Grid sistema o técnica de hojas de estilo en cascada; divide la página en una cuadrícula o rejilla con la cual se pueden llegar a posicionar diferentes elementos haciendo más facil y sencillo el diseño de la página web

Murillo Ruiz, J. Rediseño de un blog Wordpress implementando CSS GRID y PWA. Recuperado de: http://openaccess.uoc.edu/webapps/o2/bitstream/10609/80266/3/jmurillorTFG0118informe_autoevaluaci%c3%b3.pdf
